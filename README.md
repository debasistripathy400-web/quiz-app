# QuizElite - The Ultimate Knowledge Challenge 🚀

**QuizElite** is a sleek, modern, and high-performance Quiz Application built with Django. It features a glassmorphic UI, real-time timers, and a vast library of quizzes across 9 diverse categories. Whether you're a student, a tech enthusiast, or just looking for some fun, QuizElite offers a premium experience to test and expand your knowledge.

![Dashboard](https://via.placeholder.com/1200x600?text=QuizElite+Dashboard+Mockup)

## ✨ Features

- **9 Categories**: Science, History, Technology, General Knowledge, Entertainment, Compiler Design, Psychology, Sports, and Fun.
- **36+ Quizzes**: Every category includes 4 unique, 20-question quizzes.
- **Premium UI**: Glassmorphic design with smooth animations and a responsive mobile-first layout.
- **Smart Timer**: Real-time 10-minute countdown for every quiz to keep the challenge real!
- **User History**: Keep track of your scores and performance over time.
- **Secure Authentication**: Built-in Login and Sign-up system with refined visibility fixes.

## 🛠️ Technology Stack

- **Backend**: Django (Python)
- **Frontend**: Vanilla HTML5, CSS3 (Custom Glassmorphism), JavaScript (Real-time timer & UI transitions)
- **Database**: SQLite (Perfect for local development and demos)
- **Icons**: Font Awesome 6.4.0
- **Typography**: Outfit & Inter from Google Fonts

## 🚀 Quick Start

### Prerequisites
- Python 3.8+
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/debasistripathy400-web/quiz-app-
   cd quiz-app-
   ```

2. **Set up a Virtual Environment (Optional but Recommended)**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install Dependencies**
   ```bash
   pip install django
   ```

4. **Run Migrations**
   ```bash
   python manage.py migrate
   ```

5. **Start the Server**
   ```bash
   python manage.py runserver
   ```

6. **Access QuizElite**
   Open your browser and navigate to `http://120.0.0.1:8000`

## 📖 How it Works

1. **Explore**: Browse through the beautifully designed category cards.
2. **Select**: Choose a quiz that interests you (e.g., "Quantum Physics Basics" under Science).
3. **Challenge**: You have 10 minutes to answer 20 questions. The timer starts as soon as you begin!
4. **Learn**: Immediate results are shown upon completion, allowing you to track your progress in your personal history.
